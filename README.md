# NightHill

Dieses Projekt habe ich selbst erstellt und dabei ein YouTube-Tutorial als Hilfe genutzt.

Die Assets wurden von itch.io heruntergeladen und von mir bearbeitet. Ich habe die Assets selbst zusammengefuegt, und die Story sowie das Gameplay stammen von mir.

Das Spiel ist noch kurz, aber ich habe dieses Projekt als Lernmethode genutzt, um zu verstehen, wie alles funktioniert. Zurzeit entwickle ich ein neues Spiel, basierend auf den Erfahrungen aus diesem Projekt.
